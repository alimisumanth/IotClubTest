Course Objectives

a) Objectives:

  1. Identify different types of sensors
  2. Analyze different types of communication protocols
  3. Construct a hardware setup for IoT Application
  4. Identify IoT Enabling Technologies
  5. Develop IoT Application

b) Course Content

Module 1:

Introduction to IoT Concepts and Terminology of the IoT

IoT Standards

Components of IoT

Difference between Digital Signal and Analog Signal

Basic of Digital Electronics

Module 3:

Types of Sensors

Hands on basic sensors with Arduino

Hands on basic sensors with Raspberry Pi 3

Module 4:

Types of Communication Protocols

Overview of IoT Platforms

IoT Architecture

Module 5:

Introcution to Cloud Computing

Introduction to Big Data

Difference between IoT and M2M

Module 6:

Hardware Security
